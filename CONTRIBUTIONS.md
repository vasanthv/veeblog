## Veeblog Contributions

Before opening a PR, please read the following guidelines.

## Current Contribution Status

Veeblog is currently **not actively accepting contributions**.

The primary purpose of making the Veeblog source code available is:

- To allow others to use the code for personal projects or to satisfy their curiosity.
- To ensure code transparency, keeping the app honest about its actions.

### Support the Project

If you’d like to support Veeblog and its development, consider becoming a sponsor through [Github sponsors](https://github.com/sponsors/vasanthv). You can also support the project by using Veeblog and upgrading to a premium tier, or by spreading the word about Veeblog to your friends and family. Your support is greatly appreciated! 🙌
