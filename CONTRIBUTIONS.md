## Veeblog Contributions

Before opening a PR, please read the following guidelines. Veeblog is currently **not actively accepting contributions**. The primary purpose of making the Veeblog source code available is to allow others to use the code for personal projects or to satisfy their curiosity, and to ensure code transparency, keeping the app honest about its actions.

If you’d like to support Veeblog and its development, consider becoming a sponsor through [GitHub Sponsors](https://github.com/sponsors/vasanthv). You can also support the project by using Veeblog, upgrading to a premium tier, or spreading the word about Veeblog to your friends and family. Your support is greatly appreciated! 🙌
